GESTURE-CONTROLLED AR

University of California, San Diego: Electrical + Computer Enginering Masters Thesis| Committee: Pamela Cosman, Electrical + Computer Engineering; Benjamin Bratton, Visual Arts and Jurgen Schulze, Computer Science + Engineering


In 1963, Ivan Sutherland made Sketchpad, the first interactive geometry program that revolutionized the computer graphics program.Tilt Brush by Google allows users to paint life-size three-dimensional object

I created a new augmented reality technology, that goes a step further: it recognizes a user’s hand gestures and allows gestural computer-aided design. This means a person using the application creates three-dimensional visual content. Previously, interacting with augmented reality content involved tapping the screen.

Users can dynamically create solid shapes between the bounds of their hands. Rectanglular prisms within parallel hands spheres within cupped hands etc. The virtual content is carefully lined up with gestural input.

iROBOT KEYBOARD HACK

In collaboration with Yuning Hui | University of California, San Diego: Embedded Systems + Robotics| Greg Hoover, Instructor Computer Science and Engineering; Nishant Baskar, Teaching Fellow Texas Instruments

The iRobot basic movement control is implemented by socket and serial port on BBB. There are three related Python files. "irobot-PC.py" is on the PC, while ""main-BBB" and "irobot-BBB.py" are on the BBB board.

The exact position of iRobot can be described by three variables: X, Y, and H for heading. All basic movement steps can be calculated from the difference between the target position and current position.

irobot-BBB.py is a library module file written in Python. It provides all necessary interfaces to control the iRobot, encapsulating in one class called iRobot. main-BBB.py is used as a bridge between the iRobot and the socket communications. It can receive the commands from the PC and invoke the control functions of the class iRobot after parsing them. It can also return any returning data through socket. irobot-PC.py reads commands from the screen and sends it to the BBB board through socket. It receives the returning data and prints it on the screen.

mBED ARM RADAR

In collaboration with Yuning Hui | University of California, San Diego: Embedded Systems + Robotics| Greg Hoover, Instructor Computer Science and Engineering; Nishant Baskar, Teaching Fellow Texas Instruments

The mbed Board is connected to the BeagleBone Black(BBB) board through serial port(p13, p14). We wrote an ISR function for the new serial port to handle the command. The main structure of this function is switch-case.

  
First Come First Serve on left and Priority scheduler on right

When a command comes in, the ISR will first shutdown the LCD and motor events generator to ensure the priority of command execution. After the execution, the two events will be recovered or kept stopped by the Stop command.

The Start/Stop commands are executed by starting or stopping the tickers for LCD and motor event generator. The command of adjusting the heading to a specific 15 degree scope is executed by setting the two global variables for the max + min heading degrees(0-180 by default). The Printing command will send out all points stored in the array.

The socket program in PC was written in C++. It has two threads, implemented by Pthread. One is for reading commands from keyboard and sends it to the BBB through UDP socket, while the other is for receiving the data from BBB and printing on the screen.

The socket program in BBB board was written in Python. The main reason is that there is a mature serial port module called Pyserial in Python, supporting non-block reading of serial port. So there is no need for multi-threads.
The program just polls the socket and the serial port in an infinite loop, forwarding data between the mbed and the PC.

VIRTUAL REALITY iPHONE CASE

Completed for Design for Autodesk Challenge| University of California, San Diego | Design Academy at Autodesk | Student Expert Application

The cost and accessibility of virtual equipment makes it extremely limited to who can enjoy virtual experiences. Autodesk gave a workshop at UC San Diego regarding using AutoDesk Fusion 360 and product design. Before this project, I had never used CAD software for products and mechanical parts. Virtual reality is a type of digital reality–a way of using technology to create environments designed for expressly human interaction.

I set out to design an iPhone case that doubles as virtual reality glasses and that can withstand more wear than the Google Cardboard. About 60% of the population has a smart phone--hopefully these technologies can be democratized
